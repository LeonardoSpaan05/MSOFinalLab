# MSOFinalLab

Bediening en knoppen:

Run
Voert het huidige programma uit. De robot beweegt op het grid volgens de ingevoerde commando’s.

Clear
Leegt het programma, wist de trace (uitvoergeschiedenis) en zet de robot terug naar de beginpositie.

Metrics
Laat informatie zien over het programma, zoals het aantal commando’s, herhalingen en de maximale nesting-diepte.

Load Program
Hiermee kun je een eerder opgeslagen programma (tekstbestand met commando’s) laden.

Example Programs
Laadt voorbeeldprogramma’s die je direct kunt uitvoeren.

het verwijderen van acties kan je doen met rechtermuisknop.

als je iets wilt toevoegen aan een repeat moet je het op de repeat blok plaatsen dan word het gebruikt in de repeat.
